## Integrating Real-Time Data Analysis for Improved Public Transportation Efficiency and Safety in Abu Dhabi

### Overview
This project demonstrates how to integrate real-time data analysis using public transportation usage statistics and traffic accident reports in Abu Dhabi for 2024. The goal is to enhance the efficiency and safety of public transportation systems.

### Prerequisites
- Python 3.x
- Pandas library
- Matplotlib library

### Installation
1. Clone the repository:
   bash
   git clone https://github.com/YourUsername/PublicTransportAnalysisAbuDhabi.git
   
2. Navigate to the project directory:
   bash
   cd PublicTransportAnalysisAbuDhabi
   
3. Install the required libraries:
   bash
   pip install pandas matplotlib
   

### Dataset
- **Public_Transportation_Usage_Abu_Dhabi_2024.csv**: Contains statistics on public transportation usage.
- **Traffic_Accident-v3.0 (values) Abu Dhabi 2024.xlsx**: Contains records of traffic accidents.

### Usage
1. Ensure the datasets are in the project directory.
2. Run the analysis script:
   bash
   python analyze_transport_data.py
   

### Output
- Visualizations of peak public transportation usage times and traffic accidents based on weather conditions.
- Correlation analysis between passenger count and number of accidents.
